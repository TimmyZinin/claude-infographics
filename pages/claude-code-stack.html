09 / Persistent Memory

Память между сессиями

Claude не теряет контекст. MOC-файлы, handoff, логи, activity_log — всё в файлах

MEMORY.md
Главный индекс: все проекты, ссылки на MOC, ключевые факты
session_handoff.md
Снапшот текущего состояния. Обновляется при каждом действии
activity_log.md
Что делают параллельные сессии. Читается при старте
session_registry.json
Lock-файл ресурсов: Playwright, SSH, браузеры
moc-*.md
6 MOC: ai-corp, sborka, smm, clients, infra, telegram
logs/*.md
Лог на каждый проект. Формат: [HH:MM] действие — результат
task_data.js
Канбан-борд: backlog → wip → test → done
CLAUDE.md
Конституция: 19 правил, MCP, деплой, спринт-протокол
